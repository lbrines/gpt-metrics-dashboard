2. Data We Collect

2.1 Session and Metrics Data

  • Session identifiers: Unique IDs for each conversation with GPTs
  • Usage metrics: Number of messages, response time, session duration
  • Interaction content: Messages sent to GPTs and generated responses (for metrics analysis only)
  • Input and output tokens: Count of tokens processed by GPTs
  • Timestamps: Dates and times of interactions

2.2 Technical Data

  • Browser information: Type, version, preferred language
  • IP address: For security and aggregated geographic analysis
  • Performance data: Latency, API errors, service status
  • System logs: Access and error logs for maintenance

2.3 Configuration Data

  • GPT configurations: Codes, names, and descriptions of configured GPTs
  • User preferences: Dashboard settings, applied filters
  • Authentication data: API tokens (hashed), access credentials
